Инструкция - Как загрузить проект в гит. 
Установить на комп git. Качать тут https://git-scm.com/downloads
Создать на компе папку под репозитории 
Перейти в новосозданную папку. 
щелкнуть пкм и выбрать bash git here
склонировать репозиторий командой git clone https://github.com/TodyTodz/SharpGroupBegginers
Перейти в папку репозитория командой cd SharpGroupBegginers
Изменить активную ветку на FirstProject командой git checkout FirstProject
Добавить в папку файлы своего проекта. 
Добавить файлы в репозиторий: вызвать git status; с помощью команды git add * пометить все подсвеченные файлы, как отслеживаемые; снова вызвать git status и убедиться, что все файлы проекта находятся под надписью changes to commit. Если не все - добавить те, что не находятся командой git add [имя файла]
Сделать коммит командой git commit. После этого откроется выбранный вами при установке редактор, в нем В САМОМ НАЧАЛЕ файла нужно набрать сопроводительное сообщение - сообщение о том, что именно делает ваш коммит. Можно написать, что загружаете проект. После закрытия редактора коммит пройдет. 
сделать push на сервер github командой git push
